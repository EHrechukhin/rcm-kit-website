## Headline

Prepare Medicaid applications and redeterminations end to end, with human sign-off, so coverage activates faster.

---

## The Work Today

Teams manage applications and redeterminations across multiple state programs — each with its own forms, timelines, and channels. The paperwork is the job, and it moves at the speed of capacity.

---

## What the Agent Does

1. Application and redetermination package preparation
2. Eligibility and redetermination timeline tracking
3. State-specific workflow handling
4. Agency correspondence management

---

## How It Works

**Inputs:**
- Resident demographic and clinical data
- Income / asset documentation
- State program rules

**Agent Steps:**
1. Assemble package per state
2. Validate completeness
3. Track agency response
4. Manage correspondence

**HITL Gate:**  
Every application and redetermination is human-reviewed before submission given the regulatory exposure.

**Outputs:**
- Submission-ready package
- Agency correspondence log
- Redetermination calendar
